Smart data management

Modular component engine

Zerv includes a component engine to make your website modular, and make your workflow more productive.

Hello World

Start with your main HTML file. Zerv will then detect components that are required and insert them for you.

index.html

<html>
<body>
    <h1>{{app.name}}</h1>
    <script src="app.js"></script>
</body>
</html>

app.js

import Zerv from "@zerv/framework";
const app = new Zerv("#app");
app.data.app = {
    name: "Hello World"
};
app.render();

Just run your favourite build tool and load the page!
